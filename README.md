## MyTasks
Android application for task management.

### Features
- Add, edit, and delete tasks
- Task listing
- Modern interface using Jetpack Compose
- State management with ViewModel
  
### Technologies Used
- Kotlin
- Android Jetpack (Compose, ViewModel, LiveData)
- MVVM Architecture
- Hilt
- Room
